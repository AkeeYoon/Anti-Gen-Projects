# Antigravity GitHub Sync Agent

이 저장소는 로컬에서 개발된 유틸리티 및 AI 에이전트 도구들을 자동으로 분류하고 버저닝하여 업로드하는 **GitHub Sync Agent**에 의해 관리되는 자동화 저장소입니다.

## 🚀 Agent 파이프라인 소개

사용자가 로컬 `inbox/` 폴더에 새로운 도구나 변경된 코드를 넣고 에이전트에게 동기화를 요청하면, 에이전트는 다음 과정을 자동으로 수행합니다:
1. **코드 분석 및 분류**: 목적과 유사도를 판단하여 독립된 프로젝트인지, 기존 프로젝트의 버전업인지 판별
2. **버저닝 및 파일 정리**: `repositories/<프로젝트명>/<버전>/` 구조로 소스 코드를 아카이빙
3. **CHANGELOG 렌더링**: 각 프로젝트의 변경점과 배포 일자를 `CHANGELOG.md`에 자동 기록
4. **선별적 배포 (Scoped Push)**: 에이전트 구동 엔진(스크립트, 지침서 등)은 철저히 메인 저장소 뷰에서 배제하고, 오직 아카이빙된 프로젝트 파일들만 무결하게 GitHub `main` 브랜치에 루트로 동기화

## 📂 Managed Projects

현재 에이전트를 통해 관리 및 릴리즈되고 있는 프로젝트 목록입니다. 각 폴더를 클릭하면 해당 프로젝트의 버전별 소스와 변경 이력을 확인할 수 있습니다.

* [**PJArea**](./repositories/PJArea) - 빔 프로젝터 오버랩 및 해상도 계산, 패턴 제너레이터 도구 (현재 v1.1)
