============================================================
PJ Area (PJ Area) 사용 설명서
============================================================

본 문서는 다수의 빔프로젝터를 이용해 거대한 화면(Wall)을 구성할 때 필요한 
해상도 계산, 오버랩(블렌딩) 수치 도출, 그리고 렌즈 정렬용 시뮬레이션 패턴을 
생성해주는 PJ Area의 공식 매뉴얼입니다.

[시작하기]
이 프로그램은 브라우저에서 바로 동작하므로 별도 설치가 필요 없습니다.
1. PJArea_App/index.html 파일을 더블 클릭하여 웹 브라우저에서 실행합니다.
2. 외부 현장이나 USB에 담아 써야 할 경우, Portable_PJArea.html 파일 하나만 
   복사해서 쓰시면 완벽하게 동일하게 구동됩니다.

[내장 튜토리얼 (Interactive Guide)]
별도의 튜토리얼 문서 없이, 프로그램 화면 좌측 상단 로고 옆의 [Tutorial] 
버튼을 클릭하면 주요 기능 7가지에 대한 인터랙티브 가이드가 즉시 실행됩니다. 
UI가 어두워지며 단계별 조작 방법을 화면에서 직접 안내합니다.

[주요 기능 및 사용법 (Step-by-Step)]

Step 1. Physical Wall Size (물리적 타겟 공간 설정)
- 가장 먼저 영상을 투사할 실제 스크린 구조물의 크기를 입력합니다.
- Wall W / Wall H: 가로/세로 길이를 밀리미터(mm) 단위로 입력합니다. (예: 10m -> 10000)
- 하단의 Target Aspect Ratio 란에 화면 비율(예: 3.33:1)이 실시간으로 자동 계산됩니다.

Step 2. Projector Specs (개별 프로젝터 사양)
- 빔프로젝터 1대가 출력하는 해상도를 입력합니다.
- Proj Width / Height: 프로젝터 픽셀 가로/세로 (예: Full HD일 경우 1920, 1080)
- 가운데 전환 버튼: 피벗(세로 투사) 시 버튼을 누르면 가로세로 값이 뒤바뀝니다.

Step 3. Overlap Control (블렌딩/겹침 영역 제어)
- 영상끼리 겹쳐서 이어붙이는(Edge Blending) 구간의 픽셀을 지정합니다.
- 슬라이더를 움직이거나 텍스트를 클릭해 값을 입력합니다.
- 가이드 칼라 판정: 오버랩 비율(%)에 따라 컬러 배지가 변합니다. 실무에서 가장 
  안정적인 15% ~ 25% 구간(초록색 뱃지)이 되도록 맞추는 것을 권장합니다.

Step 4. Projector Array Settings (프로젝터 배열)
- 프로젝터를 가로(Columns) 몇 대, 세로(Rows) 몇 대를 나란히 둘 것인지 결정합니다.
- Lock Cols / Lock Rows 옵션:
  만약 세로 1줄로만 구성한다면 Lock Rows를 선택하고 값을 1로 유지하십시오. 
  본 프로그램의 Auto-Fit 알고리즘이 작동하여, 이상적인 오버랩(초록색) 효율을 내는 
  가장 적절한 프로젝터 대수(P)를 자동으로 계산하여 추천합니다.

Step 5. 해상도 결과 확인 (Resolution Output)
- Projector Total Output: 가상으로 배열된 프로젝터 전체가 만드는 총 픽셀 영역입니다.
- True Wall Resolution: 위 영역 중에서 실제 스크린 공간(Physical Wall)에만 딱 들어맞는 
  영상 컨텐츠 제작용 크롭 해상도입니다. 이 사이즈를 캔버스로 삼으시면 됩니다.
  * MISSING AREA 현상: 투사 영역이 실제 공간보다 모자라면, 우측 뷰포트에 
    "MISSING AREA" 빗금 경고 구역이 나타나며 해상도 텍스트가 빨갛게 변합니다.

Step 6. Viewport Decor (그리드 그래픽 꾸미기)
- 우측 레퍼런스 화면의 디자인을 제어해 빔 정렬 및 왜곡 보정에 활용합니다.
- Centers & Circles: 중앙 크로스 마커와 블렌딩 확인용 대형 원형 패턴 온오프.
- Square Grid: 물리적 비율 기반의 정확한 정사각형 격자(Thin/Thick 선 두께 제어 가능).
- Projector Info: 화면 하단 프로젝터 식별 텍스트(P1 R1, Res...) 온오프.
- Color Grid: 16x16 배열의 HSL 그라데이션 타이일과 0~F 숫자를 입혀 렌즈 왜곡(Warping)을 잡는 고급 모드.
- Project Title & Logo: 캔버스 중앙의 커스텀 프로젝트 명 기입.

Step 7. Export Pattern (테스트 패턴 저장)
- 구성이 끝난 그래픽을 실제 프로젝터에 띄울 .png 패턴 소스로 다운받습니다.
- Target 옵션 차이:
  * True Wall: 실제 벽면에 딱 떨어지는 크롭 해상도로만 추출합니다.
  * Total: 스크린 영역을 벗어나는 렌즈 빛(오버슛) 픽셀까지 전체 크기를 추출합니다.
- Blend Tint 토글: 오버랩 겹침 구역의 색상 마스킹 그래픽을 포함시킬지 여부 온오프.
